Introduccion

Que es Markdown?

Markdown es un lenguaje de marcado ligero que puede usar para agregar elementos de formato a documentos de texto sin formato. Creado por John Gruber en 2004, Markdown es ahora uno de los lenguajes de marcado más populares del mundo. Permite controlar la visualización del documento, formatear palabras en negrita o cursiva, agregar imágenes y crear listas son solo algunas de las cosas que podemos hacer con Markdown

Encabezados

Para generar encabezados se debe usar las almoadillas # uno por cada nivel.

Ejemplo:

                        # Esto es una etiqueta <h1>

                        ## Esto es una etiqueta <h2>

                        ### Esto es una etiqueta <h3>

                        #### Esto es una etiqueta <h4>

                        ##### Esto es una etiqueta <h5>

                        ###### Esto es una etiqueta <h6>

Texto

El texto se genera simplemente escribiendo asi que no es necesario ningun tipo de simbolo.

Para generar parrafos en Markdown basta con pulsar dos veces la tecla intro.

Puedes estilizar los textos con negrita o cursiva o ambas usando los asteriscos * o los guiones bajos _.

Ejemplo:

                        Puedes generar palabras en **negrita**

                        O generar letras en *italic*

                        O ambas ***cursiva y negrita***

                        O ambas ___cursiva y negrita___

Puedes generar palabras en negrita

O generar letras en italic

Citas

Para generar una cita basta con poner el simbolo mayo que > al principio de un texto.

Ejemplo:

                        > Para ser humilde se necesita grandeza. - Ernesto Sábato.
Para ser humilde se necesita grandeza. - Ernesto Sábato.

Listas

Generar listas en Markdown es mas facil que en HTML. Puedes crear listas desordenadas como ordenadas

Para crear listas utiliza asteriscos *, guiones - o numeros 1.

Para anidar listas debes hacer 4 espacios en blanco en la linea del texto que sera anidado.

Ejemplo:

                        Listas ordenadas:

                        1. Uno
                        2. Dos
                        3. Tres

                        Listas desordenadas:

                        * Empieza la linea con un asterisco.

                        Alternativa

                        - Los guiones funcionan Tambien
                        - Y si quieres listas anidadas pon 4 espacios antes del guion o asterisco.

                            - Y esto

Links

Para generar un link debes usar los corchetes [] para el texto que mostrara y los parentesis () para colocar el link.

Ejemplo

                        [Google](http://google.com).

Imagenes

Reglas horizontales

Las reglas horizontales son lineas que se utilizan para separar secciones como las etiquetas hr.

Para generarlas basta con usar tres asteriscos * o tres guiones medios - o tres guiones bajos _.

Ejemplo

                        Asteriscos

                        ***

                        Guiones medios
                        ---

                        Guiones bajos
                        ___

Codigo

Para poder crear un bloque de codigo debes encerrar el texto entre 3 backsticks ``` o virgulillas

Ejemplo

                        ```
                        let one = 1;
                        ```
                        ~~~
                        let two = 2;
                        ~~~
                    
let one = 1;
let two = 2;

Tablas

Ejemplo

En html las tablas son algo complicadas de hacer pero en Markdown es mas sencillo.

Para crear celdas solo es necesario separar los textos por barras |.

Para separar las celdas de encabezado de las celdas normales solo hay que usar unos cuantos guiones medios - continuado de una barra |.
